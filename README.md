# 陈旭述职网页版

基于原始 PPT《述职报告-人力资源陈旭(1).pptx》重构的网页版本。

## 设计方向

- 更适合电脑和手机阅读
- 视觉上偏正式、稳重、国央企专题汇报风格
- 信息结构从“逐页PPT”改为“专题化网页”

## 文件

- `index.html`
- `styles.css`
- `script.js`

## 打开方式

直接双击 `index.html` 即可打开。

如果要本地起服务：

```powershell
python -m http.server 4173
```

然后访问：

```text
http://127.0.0.1:4173/陈旭述职网页版/
```

## 重构说明

- 保留了原始 PPT 的核心事实、项目和成果
- 优化为更适合述职与竞聘场景的阅读顺序
- 强化了重点结果与主管岗位胜任力表达
